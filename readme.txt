Ferramenta de agenda telefônica online.

A solução implementada pela empresa XPTO para atender esta necessidade, trata-se de uma ferramenta simples, prática e robusta em recursos.
Onde é possível registrar dados em uma agenda telefônica online de qualquer lugar, quando a ferramenta estiver conectada a internert.
A partir de uma máquina estática ou até mesmo através de uma mobile ou samrtphone, é possível cadastrar detalhadamente seus contatos com nome, email, endereço entre outras opções integradas. 
É possível gerar rapidamente relatórios de seus contatos e exportar todo o conteúdo para excel de maneira rápida e prática.
A ferramenta possui um sistema de autenticação com usuário e senha.
É possível expandir a ferramenta, pois a mesma é interligada através de módulos e funcionalidades, o que dá elasticidade em futuras expansões.
É de fácil adaptação aos ambientes de trabalho pois roda via browser(navegador).
Para realizar a instalação, é necessário possuir um servidor apache com a versão igual ou superior a 2.2, um servidor Mysql, ter o php com versão igual ou superior a versão 5.4, ter espaço em disco e memória mínimos livres de acordo com as necessidades da ferramenta.
Para utilização é preciso apenas ter acesso a internet e um navegador intalador após o servidor estar configurado e pronto para uso.


Todos os arquivos e documentações necessárias para utilização estão no diretório Instrucoes_e_db.
Para realizar a instalação, abra o diretório Instrucoes_e_db e importe o arquivo db_agenda no seu SGBD. Feito isso todas as tabelas serão inicializadas.
Cole o diretório agenda_telefonica na raiz do seu servidor apache para os dados serem carregados corretamente.
Se precisar configure o arquivo de conexao.php para apontar o caminho correto do diretório de conexão no seu servidor.

Com tudo em mãos, agora é só aproveitar sua nova agenda online.
